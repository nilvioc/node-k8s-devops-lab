# node-k8s-devops-lab

Projeto prático para estudos de DevOps utilizando Node.js, Docker, Kubernetes e CI/CD.

## 🚀 Objetivo

Criar um laboratório real para praticar:

- Linux Server
- Git / GitHub
- Node.js
- Docker
- Kubernetes
- GitHub Actions
- Boas práticas DevOps

## 📦 Tecnologias

- Node.js 20
- Express 5
- Ubuntu Server 24.04
- GitHub

## ▶️ Como executar

npm install
npm start

Servidor roda em:

http://localhost:3000

## 🔎 Endpoints

/  
Mensagem principal

/health  
Health check da aplicação

/info  
Informações da aplicação
